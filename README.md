# UnoQLEDAnimator

A lightweight, browser-based animator for the Arduino Uno Q 8×13 LED matrix. Draw frames, preview your animation, and export ready-to-use Arduino code or a reusable JSON file — all client‑side, no server required.

**Live demo: [https://mstojke.github.io/UnoQLEDAnimator/](https://mstojke.github.io/UnoQLEDAnimator/)**

![Screenshoot](/img/demo.gif)

## Features

- 8×13 LED matrix editor — click cells to toggle LEDs on/off
- Multiple frames with navigation
- Custom frame names 
- Live C code preview for Arduino in the app
- Copy to clipboard for quick pasting into your sketch
- Export .ino — generates a complete Arduino sketch using Arduino_LED_Matrix
- Import/Export JSON — save and reload your animation, share with others
- Play preview — loop through frames in the browser

## Quick start

You have two ways to use the animator:

1) Use the live demo
- Open the app: https://mstojke.github.io/UnoQLEDAnimator/

2) Run locally
- Download or clone this repository
- Open `index.html` in your browser (double‑click is fine)
	- No build steps or dependencies are required
